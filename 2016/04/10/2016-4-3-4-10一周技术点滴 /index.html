2016.4.3-4.10一周技术点滴

A标签执行顺序

a标签里面，href，onclick与click事件哪个先执行。

上代码

<div><a href="www.baidu.com" onclick="abc()" class="but">1</a></div> function abc(){ alert(222); }; $(".but").click(function(){ alert(111); });

先执行onclick再执行click最后执行href.

前端MVC模式

M:具体逻辑实现；V:页面显示与样式展示；C:页面节点的选择，注册与控制等

上代码（一般非MVC实现方式）

<--HTML--> <div id=”test” date=”aaa”></div> <--js--> $(“#test”).click(function(){ Alert($(this).attr(“date”)); });

上代码（MVC实现方式）

<--HTML--> <div id=”test” date=”aaa”></div> <--js--> <—control--> function control(even,class,Fun,arr){ $(document).on(even,class,function(){ Fun(arr); }); }; /* @param class,选取的class; @param even,要执行的事件; @param fun,事件触发的函数; @param arr,触发函数的参数; */ <——model--> 具体的函数操作，可以放到model模块中执行，在model中，才是真正处理逻辑操作 function tipInfo(arr){ alert(arr[0]); }; 执行 control("click","#test",tipInfo,[1]);

钩子机制

让代码的逻辑更加清晰，省去大量的条件判断

上代码

var objHook={ :"10", xiaodong:"20" /* 新增加的钩子在这*/ }; function Redate(name,score){ return{ name:name, score:score } }; var xiaomingDATA=Redate("xiaoming","10"); /*获取一个对象，用变量保存*/ var xiaodongDATA=Redate("xiaodong","10"); var arr=[xiaomingDATA,xiaodongDATA];/*将该对象插进数组*/ function Hook(){ var _arry=new Array(); for(var i=0;i<arr.length;i++){ var arrdata=arr[i]; var objScore=Number(arrdata.score); var objName=arrdata.name; if(objHook[objName]){ //钩子在这，通过遍历得到对象的key值而去对应value值 objScore+=Number(objHook[objName]); _arry.push(objScore); }; }; return _arry; }; //而不用钩子机制的代码为 if(arrdata.name==”xiaoming”){} else if(arrdata.name==”xiaodong”){}
