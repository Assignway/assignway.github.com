Label FOR属性

表示Label标签要绑定的HTML元素，你点击这个标签的时候，所绑定的元素将获取焦点。

用法

<Label FOR="InputBox">姓名</Label><input ID="InputBox" type="text">

jQuery.extend()方法

功能：主要是用来拓展个全局函数啦，例如$.ajax() //对Jquery本身的属性与方法进行扩展；this指向jquery对象

jQuery.fn.extend({})方法;

功能：方法用来设置对象方法的扩展 //对Jquery的函数的方法属性进行扩展；this指向jquery的fn对象；

上代码

//给jquery添加全局函数 //如$.ajax

$.extend({ obj_1:function(a){ alert(a); } }); $.obj_1("test"); //弹出test

//给jquery的对象添加方法 //如$(“.abc”).a();为类名为abc的div调用a方法

(function(jQuery){ $.fn.extend({ a:function(){ $(this).css("background-color","red") } }); })(jQuery); //$(".abc").a(); //颜色变红

理解js的&&与|| 都会将其两边的值专为布尔值再执行

a()&&b(); 若a()为真，则执行b(),并返回b()的值；

若a()为假,则直接返回a()的值，不执行b();

a()||b(); 若a()为真，则执行a(),并返回a()的值，不执行b()；

若a()为假, 则执行b(),并返回b()的值；

&& 优先级高于 ||

上代码

alert((1 && 3 || 0) && 4); //结果4 1&&3 返回3 => 3 || 0 返回 3 => 3&&4 返回 4

Div p 与DIV >P的区别

.test p 选取class名为test的下面的所有p

包括<div class=”test”><div><p></p></div></div>

.test>p 选取class名为test的P

如<div class=”test”><p></p></div>
**不包括**<div class=”test”><div><p></p></div></div>
