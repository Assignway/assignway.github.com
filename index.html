函数作用域

上代码

`
var obj={};
obj.name=”222”;
console.log(obj); //输出222
obj.name=”333”;
console.log(obj);//输出333.obj.name的值已被改变

var a=function(){
var obj={};
obj.name="222";
console.log(obj);
};
console.log(obj);  //输出333
a();//函数里输出222.函数里面的obj.name不受外部的改变。

`

杂想

1.不要写重复的代码。任何重复的代码都可以封装成一个“工具”,但也要注意使用公共函数，导致的冲突问题。码代码之前，根据业务想清楚代码的架构。

2.一个成熟的码农，不是想到什么就写什么。要先对业务，对流程有一个整体的把握，再进行代码设计。不然到处都留坑，害人害己。

js事件冲突

出现的场景：在表单验证中，我们往往会在输入框失去焦点时触发一个blur事件，但当失去焦点后点击了一个按钮，这时blur事件和click事件就都触发了，由于js是单线程的所以就出现了问题，现在需要让blur先执行验证，然后在触发click事件。

解决方法：给按钮的click事件设置延迟执行setTimeOut(fn,100)，延迟时间的设置要大于blur事件的执行时间，这样就会在blur事件执行完后在执行click事件。

上代码

<body> <div style="width:100%;overflow:hidden;"> <input class="test" type="text" ></input> <input class="btn" type="button"></input> </div> </body> <script> $(function(){ $(".test").blur(function(){ console.log(321); }); $(".btn").click(function(){ setTimeout("alert(123)",1000); }); }) </script>

jQuery.extend()方法

jQuery.fn.extend({})方法;

上代码

//给jquery添加全局函数 //如$.ajax

$.extend({ obj_1:function(a){ alert(a); } }); $.obj_1("test"); //弹出test

//给jquery的对象添加方法 //如$(“.abc”).a();为类名为abc的div调用a方法

(function(jQuery){ $.fn.extend({ a:function(){ $(this).css("background-color","red") } }); })(jQuery); //$(".abc").a(); //颜色变红

理解js的&&与|| 都会将其两边的值专为布尔值再执行

a()&&b(); 若a()为真，则执行b(),并返回b()的值；

若a()为假,则直接返回a()的值，不执行b();

a()||b(); 若a()为真，则执行a(),并返回a()的值，不执行b()；

若a()为假, 则执行b(),并返回b()的值；

&& 优先级高于 ||

上代码

alert((1 && 3 || 0) && 4); //结果4 1&&3 返回3 => 3 || 0 返回 3 => 3&&4 返回 4

如<div class=”test”><p></p></div>
**不包括**<div class=”test”><div><p></p></div></div>

论自学能力

论自学能力

很多小企业的对应聘者的要有强大的自学能力。这个社会也需要人们需要强大的自学能力。特别对我们这些程序猿来说，自我学习能力更为重要。 笔者看来，自我学习能力与其说是能力，不如说是一种心态。

  1. 这是一种享受了解知识，充实自我的心态。
  2. 这是一种敢于一切归零的心态。作为一枚程序猿，要有一种时刻将自己知识清零，重新学习的心态。
  3. 这是一种坚持学习的心态。一时热血学习几天，很多人可以办到，可是坚持学习每天学习一个小时就很少人可以办到。
  4. 这是一种平和踏实的心态。看不懂各种框架源码，比不过人家写的代码质量，没什么大不了的，此时不会，不代表以后实现不了。
  5. 这是一种自我管理，享受时间的管理的心态。规定好自己的时间，将生命浪费在美好的事情上，会感谢自己的精彩。
  6. 当我们有一颗自我学习的心。知道万丈高楼平地起，一步一个脚印。既要抬头看路，也要低头走路。内心会充满自由。
